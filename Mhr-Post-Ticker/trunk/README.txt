== Description ==

This plugin will enable news ticker in your wordpress theme. You can use news ticker via shortcode in everywhere you want, even in theme files. 

Live Preview: [Click here](https://mhrtheme.com/plugins/mhr-post-ticker/)

Buy Premium Version: [Click here](https://www.templatemonster.com/wordpress-plugins/mhr-post-ticker-ticker-wordpress-plugin-98860.html/)

== Installation ==

1. Upload `mhr-post-ticker.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place this shortcode [mhr_post_ticker] to any page, post or in widgets
4. Place `<?php echo do_shortcode("[mhr_post_ticker]"); ?>` in your templates file
