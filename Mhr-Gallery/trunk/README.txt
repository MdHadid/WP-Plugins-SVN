== Description ==

This is wordpress gallery plugin where users can create photo gallery and video gallery that will show in specific page via shortcode or sidebar via widgets.

== Installation ==

1. Upload `mhr-gallery.php.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place this shortcode [mhr_photo_gallery] or [mhr_video_gallery] to any page, post or in widgets
4. Place `<?php echo do_shortcode("[mhr_photo_gallery]"); ?>` or `<?php echo do_shortcode("[mhr_video_gallery]"); ?>` in your templates file
